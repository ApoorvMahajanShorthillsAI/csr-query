# Table 38: Assessment of Intensity Grade Definitions

|   GRADE | If required on the AE page of the CRF, the investigator will use the adjectives MILD, MODERATE, SEVERE, or LIFE-THREATENING to describe the maximum intensity of the AE. For purposes of consistency, these intensity grades are defined as follows:   |                                                               |
|---------|--------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|---------------------------------------------------------------|
|       1 | MILD                                                                                                                                                                                                                                                   | Does not interfere with participant's usual function.         |
|       2 | MODERATE                                                                                                                                                                                                                                               | Interferes to some extent with participant's usual function.  |
|       3 | SEVERE                                                                                                                                                                                                                                                 | Interferes significantly with participant's usual function.   |
|       4 | LIFE-THREATENING                                                                                                                                                                                                                                       | Life-threatening consequences; urgent intervention indicated. |
